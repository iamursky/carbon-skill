# Section

The `section` component is documented with `heading` because `heading` uses the
nearest `section` to infer the correct heading level.

Review the
[Heading component overview](https://github.com/carbon-design-system/carbon/blob/main/packages/web-components/src/components/?path=/docs/components-heading--overview) for
usage guidance and examples.
