# Section

The `Section` component is documented with `Heading` because `Heading` uses the
nearest `Section` to infer the correct heading level.

Review the
[Heading component overview](https://github.com/carbon-design-system/carbon/blob/main/packages/react/src/components/?path=/docs/components-heading--overview) for
usage guidance and examples.
